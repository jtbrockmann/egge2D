using egge2D

To use egge2D it is important to set up eclipse for working with LWJGL. After that, if the egge2D package is copied to your project, you have to create a new class.
import egge2D.EGGE2D_Game;
import egge2D.EGGE2D_DisplaySettings;

public class YourGame extends EGGE2D_Game {
   public Game(EGGE2D_DisplaySettings Settings) {
     super(Settings);
   }

   public static void main(String[] args) {
     // set up the display parameters
     EGGE2D_DisplaySettings Settings = new EGGE2D_DisplaySettings();
     Settings.setWindowSize(640, 360);
     Settings.setOpenGLSize(640, 360);
     Settings.setFramerate(60);

     // create the game
     new Game(Settings);
   }
}
